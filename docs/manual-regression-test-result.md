# PlanPilot Local MVP 수동 회귀 테스트 결과

이 문서는 `docs/manual-test-checklist.md` 기준으로 사람이 직접 수행한 MVP 수동 회귀 테스트 결과를 기록하기 위한 문서다.

현재 문서는 결과 기록용 템플릿이며, 실제 테스트는 아직 수행하지 않았다.

## 테스트 개요

- 테스트 일자: 미수행
- 테스트 브랜치: `auto/dev-loop`
- 테스트 대상 커밋: 미수행 시 기록
- 테스트 환경: 미수행 시 기록
- 전체 결과: 미수행

결과 값 기준:

- 미수행: 아직 직접 화면 확인을 하지 않았다.
- 통과: 해당 범위의 수동 확인 항목이 모두 기대대로 동작했다.
- 실패: 하나 이상의 핵심 항목에서 회귀 또는 오류가 확인되었다.
- 보류: 환경, 데이터, 정책 미정 등으로 판정을 미룬다.

## Today 화면 확인 항목

- 전체 결과: 미수행

- [ ] 전체 업무 수가 실제 업무 개수와 일치한다.
- [ ] 완료 업무 수가 `done` 상태 업무 개수와 일치한다.
- [ ] 지난 마감 업무가 기준에 맞게 표시된다.
- [ ] 7일 이내 마감 업무가 기준에 맞게 표시된다.
- [ ] 추천 업무가 rule-based 기준에 맞게 표시된다.
- [ ] 완료 업무는 지난 마감/예정/추천 목록에서 제외된다.

## 전체 업무 화면 확인 항목

- 전체 결과: 미수행

- [ ] 업무 추가가 동작한다.
- [ ] 업무 수정이 동작한다.
- [ ] 업무 삭제 확인창에서 취소하면 업무가 유지된다.
- [ ] 업무 삭제 확인창에서 확인하면 업무가 삭제된다.
- [ ] 완료/미완료 토글이 동작한다.
- [ ] 검색이 동작한다.
- [ ] 완료 업무 표시/숨김이 동작한다.
- [ ] 프로젝트 필터가 동작한다.
- [ ] 마감일 빠른 순 정렬이 동작한다.

## 프로젝트 화면 확인 항목

- 전체 결과: 미수행

- [ ] 프로젝트 추가가 동작한다.
- [ ] 프로젝트 수정이 동작한다.
- [ ] 업무가 없는 사용자 프로젝트 삭제가 동작한다.
- [ ] 기본 프로젝트는 삭제되지 않는다.
- [ ] 업무가 연결된 프로젝트는 삭제되지 않는다.
- [ ] 프로젝트별 전체/완료/미완료 업무 수가 실제 업무 상태와 일치한다.
